Courses by Semester

Semester 1

Intro to Web Development (WEB101)

This course introduces the basics of web design and development using HTML, CSS, and JavaScript. Students will learn to create static web pages and apply design principles.

Computer Science Fundamentals (CS101)

An introduction to computer science, covering topics like algorithms, data structures, and problem-solving techniques, preparing students for advanced computing topics.

Introduction to Databases (DB101)

This course introduces database concepts, including relational databases, SQL, and basic database management. Students will learn to create and manage simple databases.

Semester 2

Advanced JavaScript (WEB201)

Building on foundational JavaScript skills, this course covers more advanced topics such as asynchronous programming, APIs, and frameworks like React.js.

Database Design & Management (DB201)

This course explores advanced database design techniques, including normalization, database security, and managing large datasets using SQL and NoSQL technologies.

Introduction to Networking (NET101)

A comprehensive introduction to computer networks, including networking hardware, protocols (TCP/IP), and network security basics. Hands-on labs are a key component.

Semester 3

Web Application Development (WEB301)

This course focuses on developing dynamic web applications using server-side languages like PHP or Node.js and integrating with databases.

Mobile Application Development (APP301)

Students learn to create mobile applications for iOS and Android platforms using development environments such as React Native or Flutter.

Data Analytics & Visualization (DA301)

An introduction to data analytics, this course covers tools and techniques for analyzing large datasets and presenting insights visually through charts and dashboards.

Semester 4

Capstone Project (CAP401)

This is a project-based course where students work on a comprehensive project that applies the knowledge gained throughout the program to solve a real-world problem.

Internship (INT401)

Students gain practical experience in the field by working with local businesses or organizations on software development, web design, or data management projects.

Cybersecurity Fundamentals (CYB401)

This course covers essential cybersecurity concepts, including encryption, firewalls, and best practices for securing networks and software systems.
